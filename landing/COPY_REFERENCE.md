## PROBLEM

**Header:** Waktu kamu habis. Bukan karena kamu malas.

Setiap hari ada tumpukan hal kecil yang menyita jam-jam terbaik kamu.

Input data. Cek dokumen. Draft email. Bikin konten. Research. Tugas-tugas yang *harus* dikerjakan — tapi tidak harus kamu yang kerjakan.

Sementara itu, bisnis tambahan yang ingin kamu bangun? Terus ditunda. Karena tidak ada waktu yang tersisa.

Bukan masalah niat. Masalahnya kamu tidak punya "tangan" yang cukup.

---

## DIFERENSIASI

**Header:** Beda dari ChatGPT, Claude, atau Gemini yang sudah kamu pakai

AI biasa itu conversation tools — kamu tanya, mereka jawab, selesai. Sesi berikutnya mulai dari nol lagi.

SohibBot bukan conversation tool. Dia execution tool.

| | ChatGPT / Claude / Gemini | SohibBot |
|---|---|---|
| Akses file di laptopmu | Tidak bisa | Bisa — baca, edit, buat, hapus |
| Ingat konteks minggu lalu | Tidak | Ya — memory permanen |
| Bisa dipakai dari HP tanpa buka laptop | Tidak | Ya — lewat Telegram |
| Jalankan tugas otomatis tanpa kamu minta | Tidak | Bisa diotomasi |
| Kalau salah, bisa benerin sendiri | Tidak | Ya — self-fix |
| Data di server orang lain | Ya | Tidak — jalan di laptopmu |

Kamu tetap bisa pakai ChatGPT untuk brainstorming atau tanya-jawab cepat. Itu memang bagusnya di sana.

Tapi kalau mau ada yang **kerjakan tugasnya** — bukan cuma kasih saran — itu yang SohibBot bisa lakukan dan mereka tidak.

---

## SOLUTION

**Header:** SohibBot jadi tangan ke-3 kamu.

Satu bot di Telegram. Jalan di laptopmu sendiri. Bisa disuruh apa saja.

Kamu instruksikan lewat chat — dia kerjakan di laptop. File, dokumen, web, konten, otomasi — semua bisa diakses dan dikerjakan dari HP kamu, tanpa buka laptop.

Ibarat punya karyawan yang:
- Tidak pernah capek
- Tidak pernah lupa instruksi sebelumnya
- Tidak pernah minta naik gaji

---

## KEMAMPUAN

**Header:** Kamu suruh apa saja — dia kerjakan.

Makin spesifik instruksimu, makin bagus hasilnya. Persis seperti karyawan baru yang cepat paham.

---

**"Kerjakan tugas kantorku"**
Input data Excel, cek dokumen, buat laporan, format file — yang biasa menyita 2–3 jam bisa selesai dalam 10 menit. Bot akses file di laptopmu langsung.

**"Buatkan konten untuk hari ini"**
Caption, artikel, email, skrip video — dia tulis dari brief singkat kamu. Yang biasa butuh 1–2 jam selesai dalam 5 menit. Kamu tinggal review dan posting.

**"Risetkan ini untuk saya"**
Ketik topiknya — bot cari dari web, baca sumbernya, rangkum yang penting, simpan ke memorinya. Besok kamu tanya lagi, dia masih ingat.

**"Otomasi [tugas berulang ini]"**
Dari generate laporan mingguan sampai jadwal posting konten — kalau bisa diinstruksikan, bisa diotomasi.

---

Kalau dia tidak tahu:
Ketik "pelajari [topik]" — bot langsung belajar dari web dan simpan ke memorinya sendiri. Besok dia sudah bisa.

Kalau dia salah:
Ketik "halu" — bot baca error-nya sendiri, identifikasi masalahnya, perbaiki, restart. Kamu tidak perlu debug apapun.

Makin lama kamu pakai, makin kenal kamu.
Memory persisten lintas hari, lintas minggu. Tidak pernah mulai dari nol lagi.

## TIME VALUE

**Header:** Berapa nilai waktu yang kamu buang setiap hari?

| Tugas | Sebelum | Sesudah SohibBot |
|---|---|---|
| Input data + cek dokumen | 2–3 jam | 10 menit |
| Buat konten / draft email | 1–2 jam | 5 menit |
| Bikin website dari nol | 2–3 hari | 1 hari, 2 website |
| Riset topik dari web | 45–60 menit | 5 menit |

Konservatif: 3 jam per hari dihemat. Per bulan: 60 jam.

Kalau kamu pakai 60 jam itu untuk bangun bisnis tambahan — riset produk, nulis konten, setup halaman jual — dalam 2 minggu pertama saja kamu sudah punya produk digital siap dijual.

Itu 3 jam per hari yang selama ini kamu buang. Atau mulai kamu pakai.

## FAQ

**Q: Harus punya Claude Pro ($20/bulan)?**
Ya. SohibBot pakai Claude Code sebagai otaknya. Tapi hitung dulu: $20 ≈ Rp 320.000/bulan. Kalau kamu hemat 3 jam/hari tugas kantor, nilai waktu yang balik jauh lebih besar dari itu. Satu produk digital Rp 49.000 terjual tujuh kali sudah nutup Claude Pro sebulan.

**Q: Laptop harus selalu nyala?**
Ya — bot jalan di laptopmu. Kebanyakan user pakai laptop kerja yang jarang dimatikan penuh. Kalau laptop sleep, bot pause. Nyala lagi, bot auto-restart sendiri.

**Q: Saya tidak bisa coding sama sekali.**
Tidak perlu. Wizard setup handle semuanya — kamu cuma jawab pertanyaan sederhana: nama bot, token Telegram, selesai. Kalau stuck, hubungi via WhatsApp — kami debug sampai bisa.

**Q: Repo-nya masih bisa di-clone gratis di GitHub. Bedanya apa?**
Versi GitHub sekarang open beta. Saat repo jadi private, akses kamu terjaga — tidak perlu clone ulang. Plus kamu dapat update otomatis, support WA, dan agent pack premium yang tidak tersedia di versi free.

**Q: Kalau tidak berhasil setup?**
Kami bantu debug via WhatsApp sampai bisa. Kalau benar-benar tidak berhasil karena masalah dari sisi kami — refund penuh, tidak ada pertanyaan.

**Q: Kapan repo jadi private?**
Setelah fase beta selesai dan produk stabil. Yang beli sekarang tidak perlu khawatir — GitHub invite dikirim saat itu, akses selamanya terjaga.
