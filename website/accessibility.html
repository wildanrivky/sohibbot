Pernyataan Aksesibilitas

Berlaku sejak: 25 Mei 2026 · Terakhir ditinjau: 25 Mei 2026

Komitmen aktif — upaya pemenuhan aksesibilitas sedang berjalan

IKASI Polban berkomitmen untuk memastikan platform ikasipolban.com dapat diakses oleh semua orang, termasuk penyandang disabilitas. Kami berupaya memenuhi standar aksesibilitas web yang berlaku secara internasional.

1. Standar yang Menjadi Acuan

Platform ini mengacu pada Web Content Accessibility Guidelines (WCAG) 2.1 Level AA yang diterbitkan oleh W3C. WCAG 2.1 mendefinisikan cara membuat konten web lebih mudah diakses oleh penyandang disabilitas.

2. Status Pemenuhan Saat Ini

Kami secara aktif mengidentifikasi dan memperbaiki hambatan aksesibilitas. Berikut status fitur aksesibilitas yang sudah diimplementasikan:

Kriteria WCAG Fitur Status
1.1.1 Non-text Content Alt text pada semua logo dan gambar fungsional ✓ Terpenuhi
1.3.1 Info and Relationships Semantic HTML: header, nav, main, footer, section, article ✓ Terpenuhi
1.4.3 Contrast (Minimum) Rasio kontras teks ≥ 4.5:1 (white-dim rgba 0.82 di atas #0D0D0D) ~ Sebagian
2.1.1 Keyboard Semua fungsi dapat dioperasikan dengan keyboard ~ Sedang diuji
2.4.1 Bypass Blocks Skip navigation link "Lewati ke konten utama" di semua halaman ✓ Terpenuhi
2.4.3 Focus Order :focus-visible CSS untuk visibilitas fokus keyboard ✓ Terpenuhi
2.4.6 Headings and Labels Hierarki heading (H1→H2→H3) yang konsisten dan deskriptif ✓ Terpenuhi
3.1.1 Language of Page lang="id" di semua halaman ✓ Terpenuhi
4.1.2 Name, Role, Value ARIA roles: navigation, list, listitem, aria-label, aria-labelledby ✓ Terpenuhi
1.2.x Captions/Audio Tidak ada konten audio/video saat ini N/A

3. Hambatan yang Diketahui

Meski telah berupaya, kami mengetahui beberapa hambatan yang masih ada dan sedang dalam proses perbaikan:

  • Direktori alumni dan bisnis — konten dinamis yang di-render via JavaScript; screen reader mungkin memerlukan refresh untuk membaca konten yang baru dimuat
  • Kontras warna pada beberapa elemen — beberapa label placeholder dan metadata sekunder masih di bawah rasio 4.5:1; sedang dalam peninjauan
  • Modal detail alumni — keyboard trap di modal sudah diimplementasikan (Escape untuk tutup), tapi focus management perlu peningkatan
  • Emoji sebagai ikon dekoratif — beberapa emoji dalam kartu bisnis dan alumni belum sepenuhnya di-hide dari screen reader dengan aria-hidden

4. Teknologi yang Digunakan

Platform ini dibangun dengan teknologi yang mendukung aksesibilitas:

  • HTML5 semantik
  • CSS3 dengan media queries untuk responsivitas
  • JavaScript vanilla (tanpa framework berat)
  • Font: Google Fonts (Playfair Display + Lato) dengan font-display: swap

Platform telah diuji pada:

  • Browser: Chrome, Safari, Firefox (versi terbaru)
  • Screen reader: VoiceOver (macOS/iOS) — pengujian parsial
  • Keyboard-only navigation: pengujian parsial

7. Rencana Peningkatan

IKASI berencana melakukan peningkatan aksesibilitas secara bertahap:

  • Q3 2026: Audit kontras warna menyeluruh dengan alat otomatis
  • Q3 2026: Perbaikan focus management di modal direktori alumni
  • Q4 2026: Penambahan aria-hidden="true" pada emoji dekoratif di seluruh halaman
  • Q4 2026: Pengujian formal dengan screen reader NVDA dan JAWS
